# song-list
A simple Spring Boot CRUD app to add/update/delete songs to/from a list using PostgreSQL. Examples of use: list of favourite or themed songs.
Update database username and password in `application.properties` for it to work. The app opens on `localhost:4200`. 

Based on a series of tutorials by <a href="https://www.youtube.com/channel/UC1Be9fnFTlcsUlejgfqag0g">JavaGuides</a>.
